--Gestion des versions.

la version est un decimal, 
 - la partie entière est la version majeur, 
 - la partie décimal la version mineur

Une version majeur identique assure la compatibilité assendente. Une spec v1.584  pourra être remonté en version v1.639.
Or une version 1.6 ne pourra pas être remonté en version 2.3
